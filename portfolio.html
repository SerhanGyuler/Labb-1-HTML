    School Program

    This project involves building a fully functional console application for a fictional school, expanding on previous lab work. The application will allow the school to manage and display various data, including teacher counts by department, student information, active courses, and employee details. It will also support salary calculations, grade overviews, and allow for safe grade entry using transactions. The program will interact with a database using ADO.NET and SQL for data retrieval and manipulation.
